Sim3D - Realtime Cloth Simulation and Physics Engine [cite: 9]

A 3D physics simulation engine developed from scratch in C++ using OpenGL. [cite: 9, 10] It features a mass-spring system for textured cloth simulation [cite: 10], advanced lighting with Phong shading [cite: 11], and efficient batch rendering. [cite: 11] The engine includes a particle system with uniform grid-based collision detection [cite: 12] and an interactive scene editor built with ImGui. [cite: 13] Object-Oriented principles were applied to ensure a modular and extensible design. [cite: 14]

DreamLab - Latent Diffusion Transformer for Text2Video Generation

This project is a smaller-scale implementation of a text-to-video generation model, inspired by OpenAI's Sora. I designed and experimented with various autoencoder architectures, focusing on minimizing quality loss through a combination of perceptual and reconstruction losses. I implemented the denoising process using a Diffusion Transformer (DiT) architecture and am currently working on incorporating spatio-temporal attention and text conditioning with CLIP embeddings.

Image Captioning using CNN + RNN on Flickr8k Dataset [cite: 15]

I built a pipeline to generate captions for images using a custom-built Convolutional Neural Network (CNN) and a Recurrent Neural Network (RNN). [cite: 15] The CNN, similar to the VGG-16 architecture, was optimized for feature extraction from images. [cite: 15] These features were then fed into a single-layer RNN to generate descriptive captions. [cite: 15] The entire model was trained from scratch on the Flickr8k dataset. [cite: 15]
